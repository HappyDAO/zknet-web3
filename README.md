# zknet web3 library
js library for zknet

### Usage
```bash

# install node、npm、yarn
brew install node
brew install npm
npm install --global yarn

# clone project
git clone https://github.com/HappyDAO/zknet-web3.git

# init
cd zknet-web3
yarn

# run demo
yarn dev

# browse url(http://127.0.0.1:8080/basic/) 
# open Dev Tools(F12 in Chrome), and view console
```
